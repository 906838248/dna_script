## ✨ 主要功能

- **多脚本支持**
  - 60级魔之楔驱离
  - 40级魔之楔驱离
  - 60级皎皎币（待完善）
  - 65级扼守
  - 机傀大乱斗
  - 深红凝珠

- **图像识别**
  - 基于OpenCV的图像识别技术
  - 自动识别游戏界面元素
  - 模板图片缓存机制，避免重复加载
  - 屏幕截图缓存，减少频繁截图操作

- **录制回放**
  - 支持键盘鼠标操作的录制与回放
  - 灵活的自定义操作流程

- **进度追踪**
  - 实时显示脚本执行进度
  - 清晰的循环次数统计
  - 通用游戏循环方法，减少代码重复

- **日志记录**
  - 详细的运行日志便于调试
  - 实时显示脚本执行状态

- **快捷键管理**
  - 自定义快捷键设置
  - 支持多种操作快捷键

- **配置自动保存**
  - 自动记录上次选择的脚本
  - 自动记录上次设置的循环次数
  - 自动记录上次使用的录制名称

## 🖥️ 运行环境与兼容性

- **操作系统**：Windows
- **游戏分辨率**：16:9 的 1920x1080 分辨率
- **Python版本**：3.11 或更高版本

## 🚀 安装指南

### 从源码运行

1. **环境要求**：确保已安装 **Python 3.11** 或更高版本

2. **克隆仓库**：
   ```bash
   git clone <repository-url>
   cd 二重螺旋
   ```

3. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

4. **运行程序**：
   ```bash
   python main.py
   ```

## 📖 使用指南

### 一、 使用前配置

在启动自动化前，请务必检查并确认以下设置：

- **分辨率**
  - 请在 **16:10 的 1920x1080** 分辨率下使用
- **按键设置**
  - 请务必使用游戏 **默认** 按键绑定
- **窗口状态**
  - 游戏窗口应保持在前台，不要最小化
- **系统状态**
  - 请勿让电脑 **熄屏** 或 **锁屏**，否则将导致程序中断
- **队友特效**
  - 建议关闭队友特效，以避免干扰脚本执行

### 二、 快速上手

1. 在「自动化脚本」标签页选择要执行的脚本
2. 设置循环次数
3. 点击「开始」按钮或按F5启动
4. 在「日志」标签页查看详细的运行日志

### 三、 录制功能

1. 在「录制功能」标签页输入录制名称
2. 选择鼠标模式（绝对坐标/相对坐标）
3. 点击「开始录制」按钮
4. 执行需要录制的操作
5. 点击「停止录制」完成录制
6. 点击「播放录制」回放录制的操作

### 四、 快捷键设置

1. 在「快捷键设置」标签页查看和修改快捷键
2. 点击「保存设置」保存修改
3. 点击「重置默认」恢复默认快捷键

### 五、 版本信息

在「版本说明」标签页可以查看：
- 当前版本信息
- 最新更新日志
- 历史更新日志
- 支持的脚本列表
- 使用说明
- 注意事项

## 🔧 技术栈

- **PySide6** - Qt界面框架
- **OpenCV** - 图像处理与识别
- **pyautogui/pydirectinput** - 自动化操作
- **keyboard** - 键盘监听
- **pynput** - 鼠标监听
